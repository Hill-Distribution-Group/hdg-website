# Hill Distribution Group Website

This is the official website for Hill Distribution Group, showcasing our team, services, and expertise in distribution solutions.

## Technologies Used

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
- Heroicons

## Getting Started

First, install the dependencies:

```bash
npm install
# or
yarn install
```

Then, run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Features

- Modern, responsive design
- Team member profiles with LinkedIn integration
- Fast page loading with Next.js
- Clean, maintainable codebase with TypeScript

## Structure

- `/src/app` - Next.js app directory with routing
- `/public` - Static assets including images
- `/src/app/people` - Team members page
